# 🎮 Simon Game

A fun and interactive memory-based game inspired by the classic **Simon Says** game. Players must remember and repeat an increasingly complex sequence of colors. The game ends when the player selects the wrong sequence.

## 🎯 How to Play

1. Press any key to start the game  
2. Watch the color sequence carefully  
3. Repeat the same sequence by clicking the buttons  
4. Each round adds a new color  
5. The game ends if you click the wrong button
